Next-gen NFT platform for artists and collectors.

- [stabilitydao/rarest-interface](https://github.com/stabilitydao/rarest-interface) - frontend application|

### Start coding

#### Setup

```
git clone https://github.com/stabilitydao/rarest.git
cd rarest
yarn
```

#### Local environment

```
npx hardhat node
```

#### Testing

```
yarn test
```

#### Linting

```
yarn lint
```

#### Coverage

```
yarn coverage
```

### Cleaning

```
yarn clean
```
